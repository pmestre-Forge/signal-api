Signal API

Momentum trading signals for AI agents. Pay per call. No signup.

x402 Protocol USDC on Base Open Source No API Keys

Endpoints

GET /signal/{ticker} $0.005
BUY/SELL/HOLD signal with confidence score. RSI, ADX, MACD, volume ratio, ATR.
GET /scan/momentum $0.01
Top momentum BUY setups from 35+ US equities, ranked by confidence.
GET /risk?tickers=X,Y,Z $0.01
Portfolio risk: volatility, max drawdown, correlation, Sharpe ratio.

How It Works

1. Agent calls GET /signal/NVDA
2. Server responds HTTP 402 + payment instructions
3. Agent wallet signs $0.005 USDC transfer on Base L2
4. Agent retries with payment proof header
5. Gets signal data as JSON

Technical Details

  • Composite score from RSI(14), ADX(14), MACD(12/26/9), Volume vs 20-day avg
  • Score maps to BUY (≥30), SELL (≤-30), HOLD (between)
  • 5-minute cache, 200-entry LRU
  • FastAPI with sync endpoints (threadpool for blocking I/O)
  • Payment: x402 protocol, USDC on Base Sepolia (testnet)
